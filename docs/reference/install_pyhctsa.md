# Download and install pyhctsa from Python into a new virtual environment

Download and install pyhctsa from Python into a new virtual environment

## Usage

``` r
install_pyhctsa(venv, python)
```

## Arguments

- venv:

  `character` specifying the name of the new virtual environment to
  create

- python:

  `character` specifying the filepath to the Python interpreter to use.
  Python 3.10 is recommended

## Value

no return value; called for side effects
